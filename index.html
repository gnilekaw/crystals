Crystals Demo

Demonstrate the crystals react components. We use a mock server powered by apiary to simulate AJAX API requests.

Note that since it's a mock server, it will always return the same data. You can still see the real data sent over in the AJAX requests via a browser's developer tools.

Artwork Price Quick Edit

Quick edit an artwork's price via a text input.

Demo

Example

ReactDOM.render(
  <ArtworkPriceQuickEdit
    artwork={artwork}
    saveUrl={saveUrl}
  />,
  document.getElementById("artwork-price-quick-edit")
);
